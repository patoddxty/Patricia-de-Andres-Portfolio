Portraits of dear friends Illustration
This project was created using photographs I took of my friends. Using Photoshop and a Wacom tablet, I created these illustrations, proposing five-tone colour palettes that capture the spirit of these people. The layout format for this project was editorial design, offering several illustration proposals to accompany articles or for use as covers. El proceso creativo parte de la observación y la interpretación personal de cada retrato, buscando ir más allá de la semejanza física para traducir rasgos de carácter, emociones y actitudes en formas y color. Cada ilustración funciona como una pieza autónoma, pero también como parte de una serie coherente, pensada para dialogar con el texto y reforzar el mensaje editorial desde lo visual.
